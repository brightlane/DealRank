In order to implement LinkConnector's Content Link Technology on your pages you will need to place this Content Link JavaScript code:

<script src='//linkconnector.com/r/cl.js?OwnerID=7949&PartnerID='></script>

on your page, filling in the appropriate values for

OwnerID - your LinkConnector (LC) CustomerID [required]
PartnerID - your unique Partner Identifier for any customers you have [optional]

There are two types of content links you can add to your page--Overlays and Mentions. Overlays will create links with overlays appearing once a user hovers over the link. The Mentions section is a row of products and offers which are mentioned on the page. The Mentions will appear on your page where the Content Link JavaScript code is placed, or wherever you place this code:

<div id='lc_mentions'></div>

Additional optional parameters may be added to affect the look and feel of the Overlays and Mentions. These are listed below. To add them, simply append the Name/Value as pairs to the script call. Here is an example that alters the Link Background Color to 'yellow':

<script src='//linkconnector.com/r/cl.js?OwnerID=7949&PartnerID=&LinksLineBackgroundColor=yellow'></script>

Notes:

1. You may use the Color Names (e.g., blue), the Hex Color Code (e.g., #0000FF), or the shorter Hex Color Code (e.g., #0FF) as Values for any ...Color Name. w3schools.com is a good resource for color names and codes.
2. The values for LinksType are Overlays, Mentions, or Both.
3. The values for LinksLineStyle are dotted, dashed, solid, double, or none.
